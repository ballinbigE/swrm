# swrm v0.9.0 — Start Work

Every card can have its own branch now.

- 🌿 **Start work on a card.** Right-click any card on the Mac board → "Start work" → swrm cuts the `sc-id/slug` branch and checks it out (or switches to it if it's already there).
- 🔁 **Closes the loop.** Start work → drag to move (auto-commit) → Push & PR. The board and your git history finally move together — and it sets up auto-marking cards done when their PR merges.
